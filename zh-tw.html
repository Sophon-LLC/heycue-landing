使用情境

為你的實際工作流程而生。

適用於所有 App。無需整合，沒有市集。

旅行故事網站

讀取你的 Google Maps 歷史紀錄，建立並上線一個完整的年度旅行回顧網站。

~2 分鐘

股票研究簡報

正在看股票代碼？說「該不該買這支」— Cue 會抓取財報、新聞，並給你結論。

~60 秒

收據 → 費用報告

掃描你的收據資料夾，擷取所有項目，並輸出成試算表和 PDF。

~45 秒

照片 → 作品集網站

挑選你最好的照片、撰寫圖說、建立作品集，並部署到 Vercel。

~2 分鐘

競爭對手研究

說出 3 個競爭對手，就能得到一份包含威脅、定價和機會的單頁簡報。

~90 秒

資料 → 儀表板

將 CSV 拖到螢幕上，說「做成儀表板」— Cue 會編寫程式碼並在 Safari 中開啟。

~60 秒

會議 → 行動項目

通話結束後，說「總結並將任務加入 Notion」— 在你關閉視窗前就完成。

~20 秒

程式碼庫 → 文件網站

說「為這個專案建立文件」— Cue 會讀取你的 repo 並發布一個完整的 API 文件網站。

~3 分鐘
