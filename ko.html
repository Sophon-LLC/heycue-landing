~90초

~20초

FAQ

궁금한 점이 있으신가요?

Cue는 Mac과 Windows용 음성 AI 에이전트입니다. 단축키를 누르고, 자연어로 명령하면, Cue가 작업 내용을 이해하고 적절한 접근 방식을 찾아 완성된 결과물을 전달합니다 — 모두 핸즈프리로. 모든 앱에서 작동하며, 설정이 필요 없습니다.

Siri는 간단한 질의를 처리하지만 워크플로를 이해하거나 다단계 작업을 실행할 수 없습니다. ChatGPT는 브라우저 탭에 있어 하던 일과 분리됩니다. Cue는 데스크톱에 상주하며, 활성 컨텍스트를 이해하고, 컴퓨터에서 직접 작업을 실행하며(파일 관리, 코드, 터미널 명령), 모든 기록을 로컬에 저장합니다.

네. Cue는 무제한 음성 받아쓰기와 일일 에이전트 크레딧으로 무료 시작할 수 있습니다. 신용카드 불필요. 파워 유저를 위한 업그레이드 옵션이 곧 출시됩니다.

모든 대화 기록은 기기에 로컬 저장됩니다 — 클라우드에 업로드되지 않습니다. 컨텍스트는 로컬에서 처리되며 어떤 서버에도 저장되지 않습니다. Apple Developer ID로 서명되어 있습니다(공증 진행 중). 유휴 시 백그라운드 프로세스가 실행되지 않습니다.

Cue는 macOS 13 Ventura 이상(Apple Silicon M1-M4 및 Intel), Windows 10(버전 1809 / 2018년 10월 업데이트) 이상, 64비트에서 작동합니다. Windows ARM64는 아직 지원되지 않습니다. 각 플랫폼별로 별도 다운로드가 제공됩니다.

응용 프로그램 폴더에서 Cue를 우클릭(또는 Control + 클릭)하고 "열기"를 선택한 뒤, 대화 상자에서 "열기"를 다시 클릭하세요. 최초 한 번만 하면 되고 이후는 정상적으로 실행됩니다. Cue는 Apple Developer ID로 서명되어 있으며, 공증은 Apple에서 진행 중입니다.

Windows에서는 Alt를 길게 눌러 받아쓰기, Alt를 두 번 탭하여 Agent 모드를 활성화합니다(Mac에서는 Fn 키). 첫 실행 시 Windows 설정 → 개인 정보 → 마이크에서 마이크 권한이 필요합니다. Agent 작업은 bash/AppleScript 대신 PowerShell 명령을 사용합니다.

Cue는 영어, 중국어, 일본어, 한국어, 스페인어, 프랑스어, 독일어 등 20개 이상의 언어로 음성 입력을 지원합니다. AI는 어떤 언어로 말하든 의도를 이해합니다.
